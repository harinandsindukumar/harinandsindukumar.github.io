## Installation

1. Clone or download this repository
2. Open `index.html` in your browser to view the portfolio
3. Customize the content in `index.html`, `styles.css`, and `script.js` as needed

## Customization

### Personal Information
Update the following sections in `index.html`:
- Name in the header section
- Japanese word and meaning
- Terminal content (position, skills, etc.)
- Project details (titles, descriptions, links)
- Contact information (email, social media links)
- GitHub stats and activity

### Styling
- Modify colors in `styles.css` using CSS variables in the `:root` selector
- Adjust animations and transitions as needed
- Update fonts if desired

### Functionality
- Update Japanese words rotation in `script.js`
- Modify theme toggle functionality in `script.js`
- Adjust terminal animations in `script.js`
